# Summit 2026 十八椅子

> 是台遊戲機，椅子拉霸機

## 功能介紹

在一般情況下，椅子會在畫面中左右移動並旋轉。

當使用者按下右下角的按鈕後，椅子會隨機改變顏色，大約在5秒內會停止改變。

適用於 g0v Summit 2026 的抽獎活動。
